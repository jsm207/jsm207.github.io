Code Functionality

This network consists of four independently run C programs: three remote servers and one client. Two of these servers are identical, with application protocols designed to sort arrays of strings, while the third server would merge the two sorted arrays into one completely sorted array. The servers each accept one argument, a port number, create and bind a socket to the specified port number, and wait for a connection request from the client. The Client must create three sockets and request connections from each server, using the hostname and the port number to specify the request. The client sends the first array of strings, specified by the user, to server 1 where they are sorted and returned, repeating the process for the second array and server 2. Upon receiving the second sorted array, the client writes both arrays into server 3, which merges the two arrays into one sorted array and sends the resulting array back to the client.

Code Functionality

Program reads a specified data file with specific information of over 40,000 named places in the United States into separate classes for each place. These objects are then stored in a hash table with simple and efficient search capabilities: user specifies a place name, is provided a list of each state in which the place exists, and prints the information of the place within the state selected by the user.
